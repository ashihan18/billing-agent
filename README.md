# 🛒 BillBot - AI Retail Billing Agent

A smart conversational billing system for retail stores built with Streamlit.

## Features
- Natural language billing (`add 2 parle g`)
- Real-time cart with GST calculation
- Inventory management
- Professional receipt generation

## How to Use
1. Type commands like:
   - `add 2 parle g and 1 surf excel`
   - `show cart`
   - `print receipt`
   - `inventory`
